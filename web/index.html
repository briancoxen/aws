AWS Serverless Architecture!

About This Page

Welcome to my website hosted on an S3 bucket. This website is using AngularJS to execute requests to the AWS API Gateway which utilizes Lambda functions. To demonstrate the Lambda function you can provide a URL of an image file in the form below. After you submit the URL, the Lambda function will download the file and store it on an S3 bucket. The image will then display on the web page served from the S3 bucket.
